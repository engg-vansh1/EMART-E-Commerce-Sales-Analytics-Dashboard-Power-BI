# EMART E-Commerce Sales Analytics Dashboard (Power BI)

## 📌 Project Overview
This project showcases an interactive Power BI dashboard designed to analyze EMART e-commerce sales data and deliver actionable business insights. The dashboard helps evaluate overall sales performance, profitability, customer behavior, product categories, and payment modes.

## 📊 Key Features
- KPI tracking for Total Sales, Total Profit, Quantity Sold, and Average Order Value  
- Quarter-wise and State-wise analysis using interactive slicers  
- Monthly profit trend visualization  
- Category and Sub-Category performance analysis  
- Payment mode distribution and customer contribution insights  

---

## 🗂️ Dataset
The project is built using structured e-commerce sales data:
- **Orders.csv** – Order date, customer details, state, and city  
- **Details.csv** – Transaction-level sales, profit, quantity, category, sub-category, and payment mode  

---

## 🛠️ Tools & Technologies

## 🎯 Objective
To transform raw e-commerce data into meaningful insights using interactive dashboards that support data-driven decision-making.
